Otra forma de hacer lo mismo

En lugar de con CSS selector se pueden usar xpath para seleccionar las partes de la pagina web que quieres bajar. Resulta que con el CSS selector vi que la xpath de la tabla es: /html/body/div[3]/div[3]/div[4]/div/table. S√≠, de momento para mi tambi√©n es pura magia.

url <- 'https://es.wikipedia.org/wiki/Anexo:Municipios_de_la_provincia_de_Teruel'

  read_html() %>%
  html_nodes(xpath='/html/body/div[3]/div[3]/div[4]/div/table') %>%
  html_table(dec = ",") 
Teruel <- Teruel[[1]]
